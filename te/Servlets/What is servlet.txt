What is a Servlet?

Servlet can be described in ways, depending on the context. many

• Servlet is technology which is a used to create a web application.

• Servlet is many an API that provides interfaces and classes including documentation.

Servlet is an interface that must be implemented for creating any Servlet.

• Servlet is class that the capabilities of the extends servers and responds

ADVANTAGES OF USING SERVLETS

• Less response time because each request runs in a separate thread.

• Servlets are scalable.

• Servlets are robust and object oriented.

• Servlets are platform independent
